## How to Use These Documents

1. Read the source document for your assigned module
2. Extract key topics, practical steps, and bullet points
3. Write theory content in the EXACT format of `app/assets/webxr/theory_content.js`
4. Write MCQs in the EXACT format of `app/assets/webxr/quiz_data.js`
5. Use NSDC QP assessment criteria as the basis for quiz questions
