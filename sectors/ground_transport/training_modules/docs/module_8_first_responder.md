# Module 8: First Responder

## "The Guardian"

**Persona**: Didi (Life-Saver) | **Duration**: 25 Mins

### Overview

Empowering the driver to be a first responder. Basic life support and emergency management.

### Key Syllabus

1. **Golden Hour**: The importance of the first 60 minutes.
2. **Emergency Numbers**: Police, Ambulance, Fire, Client Security.
3. **CPR Basics**: Hands-only CPR. "Push hard, push fast."
4. **AED Awareness**: What is an AED (Automated External Defibrillator) and how to fetch one.
5. **Scene Safety**: Assessing danger before rushing in.

### Didi's Message

*"You are not just a driver; you are a guardian. If the worst happens, you are the first hope."*
