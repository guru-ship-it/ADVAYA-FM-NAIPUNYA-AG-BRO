# Module 2: Grooming & Hygiene

## "The First Impression"

**Persona**: Didi (Coaching) | **Duration**: 10 Mins

### Overview

Details the non-negotiable standards of personal presentation and vehicle hygiene.

### Key Syllabus

1. **The Uniform**: Crisp, ironed, tucked in. Shoes polished. ID card visible.
2. **Personal Hygiene**: Breath check, deodorant, clean nails.
3. **Vehicle "Cockpit" Check**: No personal items on dash, clean mats, neutral smell (no strong perfumes/food).
4. **The "Vajra" Scan**: How the daily AI selfie check verifies these points before duty.

### Didi's Message

*"Before you start the engine, look in the mirror. Do you see a professional? Respect yourself, and the world will respect you."*
