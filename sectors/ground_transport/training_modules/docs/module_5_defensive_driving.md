# Module 5: Defensive Driving (Basic)

## "The Vajra Eye"

**Persona**: Vajra (Authoritative) | **Duration**: 15 Mins + Simulation

### Overview

The core hazard perception module. Focuses on anticipation rather than reaction. Linked to the `hazard_test_engine.py` simulation.

### Key Syllabus

1. **The 3-Second Rule**: Safe following distance.
2. **Blind Spots**: The "Kill Zones" around the car.
3. **Hazard Perception Testing**:
    - Pedestrians (Children)
    - Stray Animals (Cows/Dogs)
    - Sudden Braking
4. **The "Vajra" Metric**: Tracking eye movement/reaction time (via simulation).

### Vajra's Command
